# CommandLineTimer
A simple timer script with visual alarm.

# Installation

Fist of all Download the Script.
Then install the following dependencies:
```
sudo apt install feh
```

# Usage

```
./timer.sh <time>
# time: e.g., 12s, 1m, 1h, 1m30s. If no unit is given, it defaults to minutes.
```

Examples:
```
./timer.sh 2m
./timer.sh 30s
./timer.sh 1h30m
./timer.sh 10m30s
```
